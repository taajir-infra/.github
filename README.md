# .github

Welcome to the **taajir-infra** organization's info repository!

## About This Repository

This is a special `.github` repository that serves as the central hub for team information, community health files, and default templates for all repositories within the taajir-infra organization.

## What's Inside

This repository contains:

- **Community Health Files**: Default community standards that apply across all organization repositories
- **Organization Profile**: Information displayed on the organization's GitHub profile page
- **Workflow Templates**: Reusable GitHub Actions workflows for the organization
- **Team Documentation**: Guidelines and resources for team members

## Organization Overview

The taajir-infra organization manages infrastructure and tooling for the Taajir project.
